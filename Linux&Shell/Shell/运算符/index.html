运算符

1. 算术：​

“$((运算式))” 或 “$[运算式]” 或 "expr 运算式"

推荐：“$[运算式]”

“$((运算式))”：括号太多 不好判断 括号优先运算

expr 运算式：* 需要转义 \*

计算（2+3）* 4 的值 

 S=$[(2+3)*4]

2. 关系运算符（比较运算符）：​

1. 基本语法：​

test condition

[ condition ]（注意 condition 前后要有空格）

注意：条件非空即为 true，[ atguigu ] 返回 true，[ ] 返回 false

[ 2 = 2 ]

echo $? # 注意：等于 0 才是 true（因为判断完返回 true $? 才为0）

2. 常用判断条件：​

算术 [ ] 使用如下写法 如果是 (( )) 可以正常使用 > 等运算符

  1. 两个整数之间比较

    1. -eq 等于 =（equal）
    2. -ne 不等于 !=（not equal）
    3. -lt 小于 <（less than）
    4. -le 小于等于 <=（less equal）
    5. -gt 大于 >（greater than）
    6. -ge 大于等于 >=（greater equal）

  2. 按照文件权限进行判断

    1. -r 有读的权限（read）
    2. -w 有写的权限（write）
    3. -x 有执行的权限（execute）

  3. 按照文件类型进行判断

    1. -e 文件存在（existence）
    2. -f 文件存在并且是一个常规的文件（file）
    3. -d 文件存在并且是一个目录（directory）

3. 逻辑运算符：​

  1. && 与（单 [ ] 用 -a）

    [ xx ] && [ xx ]

    [ xx -a xx ]

  2. || 或（单 [ ] 用 -0）

    [ xx ] || [ xx ]

    [ xx -o xx ]

  3. 短路用法：

    1. && 表示前一条命令执行成功时，才执行后一条命令
    2. || 表示上一 条命令执行失败后，才执行下一条命令
