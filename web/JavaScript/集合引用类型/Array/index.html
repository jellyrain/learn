Array

是用于构造数组的全局对象，数组是类似于列表的高阶对象

1. 创建 Array 对象：​

new Array(1, 2, 3, 4) // 创建数组

2. 静态方法：​

1. from()：​

从类数组对象 或者 可迭代对象中创建一个新的数组实例（浅拷贝）

ES5: Array.prototype.concat.apply([],arguments)

2. isArray()：​

用来判断某个变量是否是一个数组对象

ES5: 数组实例.constructor === Array

3. of()：​

根据 一组参数 来创建新的数组实例，支持任意的参数数量和类

ES5: Array.prototype.concat.call([], ...arguments)

3. 实例属性：​

1. length：​

数组中的元素个数

4. 实例方法：​

1. at()：​

接收一个整数值并返回该索引的项目，允许正数和负数。负整数从数组中的最后一个项目开始倒数

2. concat()：​

用于合并两个或多个数组。此方法不会更改现有数组，而是返回一个新数组

3. copyWithin()：​

浅复制数组的一部分到同一数组中的另一个位置，并返回它，不会改变原数组的长度

4. entries()：​

返回一个新的 Array Iterator 对象，该对象包含数组中每个索引的键/值对

5. every()：​

测试一个数组内的 所有元素 是否都能通过某个指定函数的测试。它返回一个布尔值

6. fill()：​

用一个固定值填充一个数组中从起始索引到终止索引内的全部元素

7. filter()：​

创建一个新数组，其包含通过所提供函数实现的测试的所有元素

8. find()：​

返回数组中满足提供的测试函数的 第一个元素的值。否则返回 undefined

9. findIndex()：​

返回数组中满足提供的测试函数的 第一个元素的索引。若没有找到对应元素则返回 -1

10. flat()：​

按照一个可指定的深度递归遍历数组，并将所有元素与遍历到的子数组中的元素合并为一个新数组返回

扁平化：默认值为 1

使用 Infinity，可展开任意深度的嵌套数

11. flatMap()：​

使用映射函数映射每个元素，然后将结果压缩成一个新数组

12. forEach()：​

对数组的每个元素执行一次给定的函数

13. includes()：​

判断一个数组是否包含一个指定的值，如果包含则返回 true，否则返回 false

14. indexOf()：​

返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回 -1

15. join()：​

将一个数组的所有元素连接成一个字符串并返回这个字符串

16. keys()：​

返回一个包含数组中每个索引键的 Array Iterator 对象

17. lastIndexOf()：​

返回指定元素在数组中的最后一个的索引，如果不存在则返回 -1

18. map()：​

返回一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值

19. pop()：​

从数组中删除最后一个元素，并返回该元素的值

20. push()：​

将一个或多个元素添加到数组的末尾，并返回该数组的新长度

21. reduce()：​

对数组中的每个元素执行一个由您提供的 reducer 函数（升序执行），将其结果汇总为单个返回值

22. reduceRight()：​

接受一个函数作为累加器（accumulator）和数组的每个值（从右到左）将其减少为单个值

23. reverse()：​

将数组中元素的位置颠倒，并返回该数组。该方法会改变原数组

24. shift()：​

从数组中删除第一个元素，并返回该元素的值

25. slice()：​

提取源数组的一部分并返回一个新数组

26. some()：​

测试数组中是不是至少有一个元素通过了被提供的函数测试

27. sort()：​

对数组元素进行原地排序并返回此数组

28. splice()：​

通过删除或替换现有元素或者原地添加新的元素来修改数组，并以数组形式返回被修改的内容

29. toLocaleString()：​

返回一个字符串表示数组中的元素。数组中的元素将使用各自的 Object.prototype.toLocaleString() 方法转成字符串

30. toString()：​

返回一个字符串表示指定的数组及其元素。数组中的元素将使用各自的 Object.prototype.toString() 方法转成字符串

31. unshift()：​

将一个或多个元素添加到数组的头部，并返回该数组的新长度

32. values()：​

返回一个新的 Array Iterator 对象，该对象包含数组每个索引的值
