ArrayBuffer

用来表示通用的、固定长度的原始二进制数据缓冲区（用于在内存中分配特定数量的字节空间）

1. 创建 ArrayBuffer 对象：​

new ArrayBuffer(length)

length：要创建的 ArrayBuffer 的大小，单位为字节

2. 静态属性：​

1. length：​

ArrayBuffer 构造函数的 length 属性，其值为 1

2. byteLength：​

只读属性，表示 ArrayBuffer 的 byte 的大小，在 ArrayBuffer 构造完成时生成，不可改变

3. 静态方法：​

1. isView(arg)：​

如果参数是 ArrayBuffer 的视图实例则返回 true，例如 类型数组对象 或 DataView 对象；否则返回 false

4. 实例属性：​

1. byteLength：​

的只读大小（以字节为单位）ArrayBuffer。这是在构造数组时建立的，不能更改

5. 实例方法：​

1. slice()：​

返回一个 new ArrayBuffer，其内容是 thisArrayBuffer 字节从 begin（包括）到 end（不包括）的副本

如果其中一个 begin 或 end 为负数，它指的是从数组末尾开始的索引，而不是从开头开始
