Element

Element 是一个通用性非常强的基类，所有 Document 对象下的对象都继承自它。这个接口描述了所有相同种类的元素所普遍具有的方法和属性。一些接口继承自 Element 并且增加了一些额外功能的接口描述了具体的行为。

1. 实例属性：​

1. attributes 只读：​

返回一个与该元素相关的所有属性集合 NamedNodeMap

2. classList 只读：​

返回该元素包含的 class 属性，返回一个元素的类属性的实时 DOMTokenList 集合

相比将 element.className 作为以空格分隔的字符串来使用，classList 是一种更方便的访问元素的类列表的方法

DOMTokenList：​

1. 实例属性：​
1. length 只读：​

一个整数，表示存储在该对象里值的个数

2. value：​

一个字符串，返回 DOMTokenList 列表的值，和 element.className 的值相同

实例方法：​
1. item(index)：​

根据传入的索引值返回一个值，如果索引值大于等于符号列表的长度（length），则返回 undefined 或 null

2. contains(token)：​

如果 DOMTokenList 列表中包括相应的字符串 token，则返回 true，否则返回 false

3. add(token1[, token2[, ...tokenN]])：​

添加一个或多个标记（token）到 DOMTokenList 列表中

4. remove(token1[, token2[, ...tokenN]])：​

从 DOMTokenList 列表中移除一个或多个标记（token）

5. replace(oldToken, newToken)：​

使用 newToken 替换 token

6. toggle(token [, force])：​

从 DOMTokenList 字符串中移除标记字串（token），并返回 false

如果传入的字串（token）不存在，则将其添加进去，并返回 true

force 是一个可选的布尔值：

如果传入 true ，且传入的 token 不存在，则将其添加进去并返回 true ，若传入的 token 存在，则直接返回 true

如果传入 false ，则移除存在的 token ，并返回 false ，如 token 不存在则直接返回 false

7. entries()：​

返回一个迭代器（iterator），以遍历这个对象中的所有键值对

8. forEach(callback [, thisArg])：​

为每个 DOMTokenList 中的元素都调用一次传入的 callback 函数 。

9. keys()：​

返回一个迭代器（iterator）以遍历这个对象中所有键值对的键。

10. values()：​

返回一个迭代器（iterator）以遍历这个对象中所有键值对的值

3. className：​

获取或设置指定元素的 class 属性的值，可以是由空格分隔的多个 class 属性值

4. clientHeight 只读：​

对于没有定义 CSS 或者内联布局盒子的元素为 0，否则，它是元素内部的高度 (单位像素)，包含内边距，但不包括水平滚动条、边框和外边距

clientHeight 可以通过 CSS height + CSS padding * 2

5. clientLeft 只读：​

表示一个元素的左边框的宽度，以像素表示。如果元素的文本方向是从右向左（RTL, right-to-left），并且由于内容溢出导致左边出现了一个垂直滚动条，则该属性包括滚动条的宽度

6. clientTop 只读：​

一个元素顶部边框的宽度（以像素表示）。不包括顶部 外边距 或 内边距。

7. clientWidth 只读：​

该属性包括内边距 padding，但不包括边框 border、外边距 margin 和垂直滚动条（如果有的话）

clientWidth 可以通过 CSS width + CSS padding * 2

8. id：​

是一个 DOMString 表示这个元素的 id

9. innerHTML：​

设置或获取 HTML 语法表示的元素的后代。

10. localName 只读：​

是一个 DOMString 表示这个元素名称本地化的部分

11. nextElementSibling 只读：​

是一个 Element, 该元素下一个兄弟节点，如果为 null 表示不存在.

12. outerHTML：​

是一个 DOMString，获取该 DOM 元素及其后代的 HTML 文本。在设置它的时候，会从给定的字符串开始解析，替换自身

13. previousElementSibling 只读：​

是一个 Element, 该元素上一个兄弟节点，如果为 null 表示不存在

14. scrollHeight 只读：​

是一个元素内容高度的度量，包括由于溢出导致的视图中不可见内容

值等于元素在不使用水平滚动条的情况下适合视口中的所有内容所需的最小宽度。 宽度的测量方式与 clientHeight 相同：它包含元素的内边距，但不包括边框，外边距或垂直滚动条（如果存在）

15. scrollLeft：​

可以读取或设置元素滚动条到元素左边的距离

16. scrollTop：​

可以获取或设置一个元素的内容垂直滚动的像素数

17. scrollWidth 只读：​

包括由于 overflow 溢出而在屏幕上不可见的内容

值等于元素在不使用水平滚动条的情况下适合视口中的所有内容所需的最小宽度。 宽度的测量方式与 clientWidth 相同：它包含元素的内边距，但不包括边框，外边距或垂直滚动条（如果存在）

18. tagName 只读：​

返回当前元素的标签名

2. 实例方法：​

1. addEventListener()：​

将指定的监听器注册到 EventTarget 上，当该对象触发指定的事件时，指定的回调函数就会被执行

target.addEventListener(type, listener, useCapture)

target.addEventListener(type, listener, options)

1. 参数：​

  1. type：是一个 DOMString，表示事件类型
  2. listener：是一个函数，表示要注册的事件处理程序
  3. useCapture：是一个布尔值，表示事件处理程序是否可以捕获事件，默认为 false
  4. options：是一个 EventListenerOptions 对象，表示事件处理程序的选项
    1. capture：是一个布尔值，表示 listener 会在该类型的事件捕获阶段传播到该 EventTarget 时触发，默认为 false
    2. once：是一个布尔值，表示 listener 在添加之后最多只调用一次，默认为 false
    3. passive：是一个布尔值，表示 listener 永远不会调用 preventDefault()，默认为 false
    4. signal：是一个布尔值，表示 该 AbortSignal 的 abort() 方法被调用时，监听器会被移除，默认为 false

2. closest()：​

匹配特定选择器且离当前元素最近的祖先元素（也可以是当前元素本身）。如果匹配不到，则返回 null

target.closest(selectors)

1. 参数：​

selectors：是一个 DOMString，表示一个选择器

2. 返回值：​

查询到的祖先元素，也可能是 null

3. dispatchEvent()：​

向一个指定的事件目标派发一个事件, 并以合适的顺序同步调用目标元素相关的事件处理函数

!target.dispatchEvent(event)

1. 参数：​

event：是一个 Event 对象，表示要派发的事件

2. 返回值：​

当该事件是可取消的 (cancelable 为 true) 并且至少一个该事件的 事件处理方法 调用了 Event.preventDefault()，则返回值为 false；否则返回 true

4. getAttribute()：​

返回元素指定属性的属性值。如果指定的属性不存在，则返回 null 或 ''

element.getAttribute(attributeName)

1. 参数：​

attributeName：是一个 DOMString，表示要获取的属性名

2. 返回值：​

返回元素指定属性的属性值。如果指定的属性不存在，则返回 null 或 ''

5. getAttributeNames()：​

返回一个 Array，该数组包含指定元素（Element）的所有属性名称，如果该元素不包含任何属性，则返回一个空数组

6. getBoundingClientRect()：​

返回一个 DOMRect 对象，其提供了元素的大小及其相对于视口的位置

返回值：​

返回值是一个 DOMRect 对象，是包含整个元素的最小矩形（包括 padding 和 border-width）。该对象使用 left、top、right、bottom、x、y、width 和 height 这几个以像素为单位的只读属性描述整个矩形的位置和大小。除了 width 和 height 以外的属性是相对于视图窗口的左上角来计算的

7. getElementsByClassName()：​

返回一个即时更新的（live）HTMLCollection，包含了所有拥有指定 class 的子元素

element.getElementsByClassName(names)

参数：​

names：是一个字符串，表示要匹配的类名列表，类名通过空格分隔

返回值：​

一个实时集合，包含了找到的所有元素

8. getElementsByTagName()：​

返回一个动态的包含所有指定标签名的子元素的 HTML 集合

element.getElementsByTagName(name)

参数：​

name：是一个代表元素的名称的字符串。特殊字符 "*" 代表了所有元素

返回值：​

是一个由发现的元素出现在树中的顺序构成的动态的 HTML 集合

9. hasAttribute()：​

返回一个布尔值，指示该元素是否包含有指定的属性（attribute）

element.hasAttribute(attName)

参数：​

attName：是一个字符串，表示属性的名称

返回值：​

布尔值：true 或 false

10. insertAdjacentElement()：​

将一个给定的元素节点插入到相对于被调用的元素的给定的一个位置

element.insertAdjacentElement(position, element)

参数：​

  1. position：是一个字符串，表示插入的位置。可能的值有：
    1. beforebegin：在该元素本身的前面
    2. afterbegin：只在该元素当中，在该元素第一个子孩子前面
    3. beforeend：只在该元素当中，在该元素最后一个子孩子后面
    4. afterend：在该元素本身的后面
  2. element：是一个元素节点，表示要插入的元素

返回值：​

插入的元素，插入失败则返回 null

11. insertAdjacentHTML()：​

将指定的文本解析为 Element 元素，并将结果节点插入到 DOM 树中的指定位置

它不会重新解析它正在使用的元素，因此它不会破坏元素内的现有元素。这避免了额外的序列化步骤，使其比直接使用 innerHTML 操作更快

element.insertAdjacentHTML(position, text)

参数：​

  1. position：是一个字符串，表示插入的位置。可能的值有：
    1. beforebegin：在该元素本身的前面
    2. afterbegin：只在该元素当中，在该元素第一个子孩子前面
    3. beforeend：只在该元素当中，在该元素最后一个子孩子后面
    4. afterend：在该元素本身的后面
  2. text：是要被解析为 HTML 或 XML 元素，并插入到 DOM 树中的 DOMString

12. insertAdjacentText()：​

将一个给定的文本节点插入在相对于被调用的元素给定的位置

element.insertAdjacentText(position, element)

参数：​

  1. position：是一个字符串，表示插入的位置。可能的值有：
    1. beforebegin：在该元素本身的前面
    2. afterbegin：只在该元素当中，在该元素第一个子孩子前面
    3. beforeend：只在该元素当中，在该元素最后一个子孩子后面
    4. afterend：在该元素本身的后面
  2. element：是一个文本节点，表示要插入的文本

13. matches()：​

如果元素被指定的选择器字符串选择，返回 true; 否则返回 false

element.matches(selectorString)

参数：​

selectorString：是一个字符串，表示一个选择器

返回值：​

元素被指定的选择器字符串选择，返回 true; 否则返回 false

14. querySelector()：​

返回文档中与指定选择器或选择器组匹配的第一个 后代 Element 对象。 如果找不到匹配项，则返回 null

element.querySelector(selectors)

参数：​

selectors：包含一个或多个要匹配的选择器的 DOM 字符串，该字符串必须是有效的 CSS 选择器字符串

返回值：​

表示文档中与指定的一组 CSS 选择器匹配的第一个元素，一个 Element 对象。如果没有匹配到，则返回 null

15. querySelectorAll()：​

返回与指定的选择器组匹配的文档中的 后代元素 列表 (使用深度优先的先序遍历文档的节点)。返回的对象是 NodeList

element.querySelectorAll(selectors)

参数：​

selectors：包含一个或多个要匹配的选择器的 DOM 字符串，该字符串必须是有效的 CSS 选择器字符串

返回值：​

一个静态 NodeList，包含一个与至少一个指定选择器匹配的元素的 Element 对象，或者在没有匹配的情况下为空 NodeList

16. remove()：​

把对象从它所属的 DOM 树中删除

17. removeAttribute()：​

从指定的元素中删除一个属性

element.removeAttribute(attrName)

参数：​

attrName：指定要从元素中移除的属性的名称

18. removeEventListener()：​

删除使用 EventTarget.addEventListener() 方法添加的事件。使用事件类型，事件侦听器函数本身，以及可能影响匹配过程的各种可选择的选项的组合来标识要删除的事件侦听器

target.removeEventListener(type, listener[, useCapture])

target.removeEventListener(type, listener[, options])

参数：​

  1. type：是一个 DOMString，表示事件类型
  2. listener：需要从目标事件移除的事件监听函数
  3. useCapture：是一个布尔值，表示事件处理程序是否可以捕获事件，默认为 false
  4. options：是一个 EventListenerOptions 对象，表示事件处理程序的选项
    1. capture：是一个布尔值，表示这个类型的事件将会被派遣到已经注册的侦听器，然后再派遣到 DOM 树中它下面的任何 EventTarget，默认为 false

19. scroll()：​

是用于在给定的元素中滚动到某个特定坐标的 Element 接口

element.scroll(x_coord, y_coord)

element.scroll(options)
element.scroll({
  top: 100,
  left: 100,
  behavior: 'smooth',
})

参数：​

  1. x_coord：是指在元素左上方区域横轴方向上想要显示的像素
  2. y_coord：是指在元素左上方区域纵轴方向上想要显示的像素
  3. options：是一个 ScrollToOptions 的字典
    1. behavior：指定滚动是平滑动画（smooth），还是在一次跳转中立即发生（auto，默认值）

20. scrollBy()：​

是使得元素滚动一段特定距离的 Element 接口 Scrolls an element by the given amount

element.scrollBy(x_coord, y_coord)

element.scrollBy(options)
element.scrollBy({
  top: 100,
  left: 100,
  behavior: 'smooth',
})

参数：​

  1. x_coord：是指在元素左上方区域横轴方向上想要显示的像素
  2. y_coord：是指在元素左上方区域纵轴方向上想要显示的像素
  3. options：是一个 ScrollToOptions 的字典
    1. behavior：指定滚动是平滑动画（smooth），还是在一次跳转中立即发生（auto，默认值）

21. scrollTo()：​

可以使界面滚动到给定元素的指定坐标位置

element.scrollTo(x_coord, y_coord)

element.scrollTo(options)
element.scrollTo({
  top: 100,
  left: 100,
  behavior: 'smooth',
})

参数：​

  1. x_coord：是指在元素左上方区域横轴方向上想要显示的像素
  2. y_coord：是指在元素左上方区域纵轴方向上想要显示的像素
  3. options：是一个 ScrollToOptions 的字典
    1. behavior：指定滚动是平滑动画（smooth），还是在一次跳转中立即发生（auto，默认值）

22. setAttribute()：​

设置指定元素上的某个属性值。如果属性已经存在，则更新该值；否则，使用指定的名称和值添加一个新的属性

element.setAttribute(name, value)

参数：​

  1. name：是一个 DOMString，表示属性名称
  2. value：是一个 DOMString，表示属性值

23. toggleAttribute()：​

切换给定元素的某个布尔值属性的状态（如果属性不存在则添加属性，属性存在则移除属性）

element.toggleAttribute(name [, force]);

参数：​

  1. name：是一个 DOMString，表示属性名称
  2. force：是一个布尔值，表示是否强制切换属性的状态

3. HTMLElement 接口：​

表示所有的 HTML 元素。一些 HTML 元素直接实现了 HTMLElement 接口，其它的间接实现 HTMLElement 接口
