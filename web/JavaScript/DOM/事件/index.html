事件

1. 注册事件方法：​

1. 方法一：​

element.addEventListener('click', e => {
  console.log('click')
})

2. 方法二：​

需要在事件名前加上 on 前缀，如 onclick

element.onclick = e => {
  console.log('click')
}

3. 方法三：​

需要在事件名前加上 on 前缀，如 onclick

<div onclick="handleClick()"></div>

<script>
  function handleClick(e) {
    console.log('click')
  }
</script>

2. 事件：​

1. FocusEvent：​

  • blur：失去焦点时触发
  • focus：获得焦点时触发

2. UiEvent：​

  • abort：加载中止时触发
  • error：加载失败时触发
  • load：加载完成时触发
  • resize：调整浏览器窗口大小时触发
  • scroll：滚动条滚动时触发
  • select：用户选择文本后触发 (input 和 textarea)

3. InputEvent：​

  • input：元素值发生改变时触发

4. Event：​

  • change：失去焦点且内容发生改变时触发

5. MouseEvent：​

  • click：左键单击元素时触发
  • dblclick：左键双击元素时触发
  • contextmenu：右键单击时触发
  • mousedown：元素上按下鼠标按钮时触发
  • mouseup：元素上释放鼠标按钮时触发
  • mouseenter：移到元素上时触发
  • mouseleave：移出元素时触发
  • mousemove：元素上移动时触发

6. KeyboardEvent：​

  • keydown：按下按键时触发
  • keypress：按下按键时触发 (功能键不触发)
  • keyup：释放按键时触发

7. ClipboardEvent：​

  • copy：复制元素内容时触发
  • cut：剪切元素内容时触发
  • paste：粘贴内容到元素中触发

8. DragEvent：​

  • drag：拖动元素时触发
  • dragend：拖动操作结束时触发
  • dragenter：拖动元素移到目标时触发
  • dragleave：拖动元素移出目标时触发
  • dragover：拖动元素位于目标上时触发
  • dragstart：开始拖动元素时触发
  • drog：拖动元素放置到目标上触发

9. WheelEvent：​

  • wheel：鼠标滚轮滚动时触发
