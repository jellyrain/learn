Reflect

1. 反射机制描述：​

反射机制指的是程序在运行时能够获取自身的信息

2. Reflect 简述：​

  1. Reflect 是一个内建的对象，用来提供方法去拦截 JavaScript 的操作（封装原始行为）
  2. Reflect 不是一个函数对象，所以它是不可构造的，也就是说它不是一个构造器，你不能通过new操作符去新建或者将其作为一个函数去调用 Reflect 对象
  3. Reflect 的所有属性和方法都是静态的
  4. Reflect 内部封装了一系列对象的底层操作
  5. Reflect 成员方法就是 Proxy 处理对象的默认实现

3. Reflect 设计目的：​

1. 将 Object 对象的一些明显属于语言内部的方法（比如 Object.defineProperty），放到 Reflect 对象上：​

现阶段，某些方法同时在 Object 和 Reflect 对象上部署，未来的新方法将只部署在 Reflect 对象上。也就是说，从 Reflect 对象上可以拿到语言内部的方法

2. 修改某些 Object 方法的返回结果，让其变得更合理：​

比如，Object.defineProperty(obj, name, desc) 在无法定义属性时，会抛出一个错误，而 Reflect.defineProperty(obj, name, desc) 则会返回 false

3. 让 Object 操作都变成函数行为：​

某些 Object 操作是命令式，比如 name in obj 和 delete obj[name]，而 Reflect.has(obj, name) 和 Reflect.deleteProperty(obj, name) 让它们变成了函数行为

4. Reflect 对象的方法与 Proxy 对象的方法一一对应，只要是 Proxy 对象的方法，就能在 Reflect 对象上找到对应的方法：​

这就让 Proxy 对象可以方便地调用对应的 Reflect 方法，完成默认行为，作为修改行为的基础。也就是说，不管 Proxy 怎么修改默认行为，你总可以在 Reflect 上获取默认行为

4. 静态方法：​

1. get(target, propertyKey, receiver)：​

与从对象 target[propertyKey] 中读取属性类似，但它是通过一个函数执行来操作的

Reflect.get(target, propertyKey[, receiver])

1. 参数描述：​

  1. target：目标对象
  2. propertyKey：需要获取的值的键值
  3. receiver：如果 target 对象中指定了 getter，receiver 则为 getter 调用时的 this 值

2. 返回值：​

属性的值

3. 代替方式：​

target[propertyKey]

2. set(target, propertyKey, value, receiver)：​

就像在一个对象上设置一个属性

Reflect.set(target, propertyKey, value[, receiver])

1. 参数描述：​

  1. target：设置属性的目标对象
  2. propertyKey：设置的属性的名称
  3. value：设置的值
  4. receiver：如果遇到 setter，receiver 则为 setter 调用时的 this 值

2. 返回值：​

返回一个 Boolean 值表明是否成功设置属性

3. 代替方式：​

target[propertyKey] = value

3. has(target, propertyKey)：​

与 in 操作符 相同

Reflect.has(target, propertyKey)

1. 参数描述：​

  1. target：目标对象
  2. propertyKey：属性名，需要检查目标对象是否存在此属性

2. 返回值：​

一个 Boolean 类型的对象指示是否存在此属性

3. 代替方式：​

propertyKey in target

4. deleteProperty(target, propertyKey)：​

与 in 操作符 相同

Reflect.deleteProperty(target, propertyKey)

1. 参数描述：​

  1. target：删除属性的目标对象
  2. propertyKey：需要删除的属性的名称

2. 返回值：​

Boolean 值表明该属性是否被成功删除

3. 代替方式：​

delete target[propertyKey]

5. construct(target, argumentsList, newTarget)：​

有点像 new 操作符 构造函数

Reflect.construct(target, argumentsList[, newTarget])

1. 参数描述：​

  1. target：被运行的目标构造函数
  2. argumentsList：类数组，目标构造函数调用时的参数
  3. newTarget：作为新创建对象的原型对象的 constructor 属性， 参考 new.target 操作符，默认值为 target

2. 返回值：​

以 target（如果 newTarget 存在，则为 newTarget）函数为构造函数，argumentList 为其初始化参数的对象实例

3. 代替方式：​

new target(...argumentsList)

6. getPrototypeOf(target)：​

与 Object.getPrototypeOf() 方法几乎是一样的。都是返回指定对象的原型（即内部的 [[Prototype]] 属性的值）

Reflect.getPrototypeOf(target)

1. 参数描述：​

  1. target：获取原型的目标对象

2. 返回值：​

给定对象的原型。如果给定对象没有继承的属性，则返回 null

3. 代替方式：​

Object.getPrototypeOf(target)

7. setPrototypeOf(target, prototype)：​

与 Object.setPrototypeOf() 方法几乎是一样的。它可设置对象的原型（即内部的 [[Prototype]] 属性）为另一个对象或 null，如果操作成功返回 true，否则返回 false

Reflect.setPrototypeOf(target, prototype)

1. 参数描述：​

  1. target：获取原型的目标对象
  2. prototype：对象的新原型（一个对象或 null）

2. 返回值：​

返回一个 Boolean 值表明是否原型已经成功设置

3. 代替方式：​

Object.setPrototypeOf(target, prototype)

8. apply(target, prototype)：​

通过指定的参数列表发起对目标 (target) 函数的调用

Reflect.apply(target, thisArgument, argumentsList)

1. 参数描述：​

  1. target：目标函数
  2. thisArgument：调用时绑定的 this 对象
  3. argumentsList：调用时传入的实参列表，该参数应该是一个类数组的对象

2. 返回值：​

返回值是调用完带着指定参数和 this 值的给定的函数后返回的结果

3. 代替方式：​

target.apply(thisArgument, argumentsList)

9. defineProperty(target, propertyKey, attributes)：​

基本等同于 Object.defineProperty() 方法，唯一不同是返回 Boolean 值

Reflect.defineProperty(target, propertyKey, attributes)

1. 参数描述：​

  1. target：目标对象
  2. propertyKey：要定义或修改的属性的名称
  3. attributes：要定义或修改的属性的描述符（具体查看 Object.defineProperty 部分）

2. 返回值：​

Boolean 值指示了属性是否被成功定义

3. 代替方式：​

Object.defineProperty(target, propertyKey, attributes)

10. getOwnPropertyDescriptor(target, propertyKey)：​

与 Object.getOwnPropertyDescriptor() 方法相似。如果在对象中存在，则返回给定的属性的属性描述符。否则返回 undefined

Reflect.getOwnPropertyDescriptor(target, propertyKey)

1. 参数描述：​

  1. target：需要寻找属性的目标对象
  2. propertyKey：获取自己的属性描述符的属性的名称

2. 返回值：​

如果属性存在于给定的目标对象中，则返回属性描述符；否则，返回 undefined

3. 代替方式：​

Object.getOwnPropertyDescriptor(target, propertyKey)

11. isExtensible(target)：​

判断一个对象是否可扩展（即是否能够添加新的属性）。与它 Object.isExtensible() 方法相似

Reflect.isExtensible(target)

1. 参数描述：​

  1. target：检查是否可扩展的目标对象

2. 返回值：​

返回一个 Boolean 值表明该对象是否可扩展

3. 代替方式：​

Object.isExtensible(target)

12. preventExtensions(target)：​

阻止新属性添加到对象 （例如：防止将来对对象的扩展被添加到对象中）。与它 Object.preventExtensions() 方法相似

Reflect.preventExtensions(target)

1. 参数描述：​

  1. target：阻止扩展的目标对象

2. 返回值：​

返回一个 Boolean 值表明目标对象是否成功被设置为不可扩展

3. 代替方式：​

Object.preventExtensions(target)

13. ownKeys(target)：​

返回一个由目标对象自身的属性键组成的数组

Reflect.ownKeys(target)

1. 参数描述：​

  1. target：获取自身属性键的目标对象

2. 返回值：​

由目标对象的自身属性键组成的 Array

3. 代替方式：​

Object.keys(target)
