Proxy

Proxy 可以理解成，在目标对象之前架设一层“拦截”，外界对该对象的访问，都必须先通过这层拦截，因此提供了一种机制，可以对外界的访问进行过滤和改写

这个词的原意是代理，用在这里表示由它来“代理”某些操作，可以译为“代理器”

1. 创建 Proxy 代理对象：​

new Proxy(target, handler)
  1. target：

要使用 Proxy 包装的目标对象（可以是任何类型的对象，包括原生数组，函数，甚至另一个代理）

  1. handler：

一个通常以函数作为属性的对象，各属性中的函数分别定义了在执行各种操作时 代理 的行为

2. 静态方法：​

1. revocable(target, handler)：​

创建一个可撤销的 Proxy 对象

Proxy.revocable(target, handler)
  • 该方法的返回值是一个对象，其结构为： {"proxy": proxy, "revoke": revoke}
  1. proxy：

表示新生成的代理对象本身，和用一般方式 new Proxy(target, handler) 创建的代理对象没什么不同，只是它可以被撤销掉

  1. revoke：

撤销方法，调用的时候不需要加任何参数，就可以撤销掉和它一起生成的那个代理对象

注意：

  1. 一旦某个代理对象被撤销，它将变得几乎完全不可调用，在它身上执行任何的可代理操作都会抛出 TypeError 异常（注意，可代理操作一共有 14 种，执行这 14 种操作以外的操作不会抛出异常）
  2. 一旦被撤销，这个代理对象便不可能被直接恢复到原来的状态，同时和它关联的目标对象以及处理器对象都有可能被垃圾回收掉
  3. 再次调用撤销方法 revoke() 则不会有任何效果，但也不会报

3. handler 对象的方法：​

1. get(target, propKey, receiver)：​

用于拦截对象的 读取属性 操作

var p = new Proxy(target, {
  get: function (target, property, receiver) {
    return Reflect.get(target, propKey, receiver)
  },
})

1. 参数描述：​

  1. target：目标对象
  2. property：被获取的属性名
  3. receiver：Proxy 或者继承 Proxy 的对象

2. 拦截目标：​

  1. 访问属性：proxy[foo] 和 proxy.bar
  2. 访问原型链上的属性：Object.create(proxy)[foo]（get 方法可以继承）
  3. Reflect.get()

3. 返回值：​

  1. 可以返回任何值

2. set(target, property, value, receiver)：​

用于拦截对象的 设置属性值 操作

const p = new Proxy(target, {
  set: function (target, property, value, receiver) {
    return Reflect.set(target, propKey, value, receiver)
  },
})

1. 参数描述：​

  1. target：目标对象
  2. property：将被设置的 属性名 或 Symbol
  3. value：新属性值
  4. receiver：Proxy 或者继承 Proxy 的对象

2. 拦截目标：​

  1. 指定属性值：proxy[foo] = bar 和 proxy.foo = bar
  2. 指定继承者的属性值：Object.create(proxy)[foo] = bar（set 方法可以继承）
  3. Reflect.set()

3. 返回值：​

  1. 返回 true 代表属性设置成功
  2. 在严格模式下，如果 set() 方法返回 false，那么会抛出一个 TypeError 异常

3. apply(target, thisArg, argumentsList)：​

用于拦截函数的调用

var p = new Proxy(target, {
  apply: function (target, thisArg, argumentsList) {
    return Reflect.apply(target, thisArg, argumentsList)
  },
})

1. 参数描述：​

  1. target：目标对象（函数）
  2. thisArg：被调用时的上下文对象
  3. argumentsList：被调用时的参数数组

2. 拦截目标：​

  1. proxy(...args)
  2. Function.prototype.apply() 和 Function.prototype.call()
  3. Reflect.apply()

3. 返回值：​

可以返回任何值

4. has(target, prop)：​

是针对 in 操作符的代理方法

var p = new Proxy(target, {
  has: function (target, prop) {
    return Reflect.has(target, prop)
  },
})

1. 参数描述：​

  1. target：目标对象
  2. prop：需要检查是否存在的属性

2. 拦截目标：​

  1. 属性查询：foo in proxy
  2. 继承属性查询：foo in Object.create(proxy)
  3. with 检查: with(proxy) { (foo); }
  4. Reflect.has()

3. 返回值：​

返回一个 boolean 属性的值

5. construct(target, argumentsList, newTarget)：​

用于拦截 new 操作符

var p = new Proxy(target, {
  construct: function (target, argumentsList, newTarget) {
    return Reflect.construct(target, argumentsList, newTarget)
  },
})

1. 参数描述：​

  1. target：目标对象
  2. argumentsList：constructor 的参数列表
  3. newTarget：最初被调用的构造函数

2. 拦截目标：​

  1. new proxy(...args)
  2. Reflect.construct()

3. 返回值：​

必须返回一个对象
