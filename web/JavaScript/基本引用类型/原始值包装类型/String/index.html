String

一个用于字符串或一个字符序列的构造函数

1. 创建 String 对象：​

String(thing)
new String(thing)

thing：任何可以被转换成字符串的值

3. 静态方法：​

1. fromCharCode()：​

通过一串 Unicode 创建字符串（ES5 不能识别大于 32 位）

2. fromCodePoint()：​

通过一串 码点 创建字符串（ES6 能识别大于 32 位）

3. raw()：​

通过模板字符串创建字符串

4. 实例属性：​

1. length：​

字符串中的 字符元素 个数

4. 实例方法：​

1. at(index)：​

接收一个整数值并返回该索引的 字符元素，允许正数和负数

负整数从字符串中的最后一个 字符元素 开始倒数

2. charAt(index)：​

接收一个整数值并返回该索引的 字符元素，允许正数

3. charCodeAt(index)：​

接收一个整数值并返回该索引的 Unicode 码值

如果指定的 index 小于 0 、等于 或 大于 字符串的长度，则 charCodeAt 返回 NaN

处理不了 Unicode 扩展编码字符（32 位 2 进制）

4. codePointAt(pos)：​

接收一个整数值并返回该索引的 Unicode 码值

如果指定的 index 小于 0 、等于 或 大于 字符串的长度，则 charCodeAt 返回 NaN

可以处理 Unicode 扩展编码字符（32 位 2 进制）

5. concat(str [, ...strN ])：​

用于合并两个或多个字符串。此方法不会更改现有字符串，而是返回一个新字符串

6. includes(searchString [, position])：​

确定调用字符串是否包含 searchString

7. endsWith(searchString [, length])：​

确定字符串是否以 searchString 字符串的字符结尾

8. indexOf(searchValue [, fromIndex])：​

String 返回调用对象中第一次出现 的索引 searchValue，或者 -1 如果没有找到

9. lastIndexOf(searchValue [, fromIndex])：​

String 返回最后一次出现的调用对象中 的索引 searchValue，或者 -1 如果没有找到

10. localeCompare(compareString [, locales [, options]])：​

返回一个数字，指示引用字符串 compareString 是在给定字符串之前、之后还是等价于排序顺序

11. match(regexp)：​

用于将正则表达式 regexp 与字符串匹配

12. matchAll(regexp)：​

regexp 返回所有匹配的迭代器

13. normalize([form])：​

返回调用字符串值的 Unicode 规范化形式

14. padEnd(targetLength [, padString])：​

用一个字符串填充当前字符串（如果需要的话则重复填充），返回填充后达到指定长度的字符串。从当前字符串的末尾（右侧）开始填充

targetLength：指定长度

padString：填充字符（默认值为空格）

15. padStart(targetLength [, padString])：​

方法用另一个字符串填充当前字符串 (如果需要的话，会重复多次)，以便产生的字符串达到给定的长度。从当前字符串的左侧开始填充

targetLength：指定长度

padString：填充字符（默认值为空格）

16. repeat(count)：​

count 返回由对象元素重复次数 组成的字符串

17. replace(searchFor, replaceWith)：​

用 replaceWith 替换出现的 searchForusing

searchFor：可以是字符串或正则表达式

replaceWith：也可以是字符串或函数

18. replaceAll(searchFor, replaceWith)：​

用 replaceWith 替换所有出现的 searchForusing

searchFor：可以是字符串或正则表达式

replaceWith：也可以是字符串或函数

19. search(regexp)：​

搜索正则表达式 regexp 和 调用字符串之间的匹配项

注意：/g 也只匹配一个 需要匹配多个 使用 match

20. slice(beginIndex[, endIndex])：​

提取某个字符串的一部分，并返回一个新的字符串，且不会改动原字符串

21. split([sep [, limit] ])：​

使用 指定的分隔符字符串 将一个 String 对象分割成子字符串数组，以一个指定的分割字串来决定每个拆分的位置

22. startsWith(searchString [, length])：​

确定调用字符串是否以 searchString 字符串的字符开头

23. substring(indexStart [, indexEnd])：​

返回一个字符串在开始索引到结束索引之间的一个子集, 或从开始索引直到字符串的末尾的一个子集

24. toLocaleLowerCase([locale, ...locales])：​

字符串中的字符在遵守当前语言环境的同时转换为小写

25. toLocaleUpperCase([locale, ...locales])：​

字符串中的字符在遵守当前语言环境的同时转换为大写

26. toLowerCase()：​

返回转换为小写的调用字符串值

27. toString()：​

返回表示指定对象的字符串。覆盖 Object.prototype.toString() 方法

28. toUpperCase()：​

返回转换为大写的调用字符串值

29. trim()：​

从字符串的开头和结尾修剪空格。ECMAScript 5 标准的一部分

30. trimStart()：​

从字符串的开头修剪空格，trimLeft() 是这个方法的别名

31. trimEnd()：​

从字符串末尾修剪空白，trimRight() 是这个方法的别名

32. valueOf()：​

返回指定对象的原始值。覆盖 Object.prototype.valueOf() 方法

5. slice 和 substring 和 substr 的区别：​

1. 不传参数：如果不传参数，默认结果为本身​

  1. slice 可用于数组截取
  2. substring、substr 只能用于字符串截取

2. 1 个参数：​

  1. substring 传一个参数且为负值时结果为 str 本身
  2. slice、substr 会与自身 length 相加，值为正，则正常运算，值为负，则结果为 str 本身

3. 2 个参数：（第一个为正）：​

  1. slice 第二个参数为负值时，会与 length 相加 还为负值的时候，为 0 ，截取的是 ''
  2. substring 第二个参数为负值时候会记为 0， 并且会对参数默认进行排序。（小的在前）
  3. substr 第二个参数是截取的位数，为负值的时候，截取的是 ''

4. 2 个参数：（第一个为负）：​

  1. slice 第 1 个参数为负值时，会与 length 相加 还为负值的时候，为 0 ，然后运算
  2. substring 第 1 个参数为负值时候会记为 0
  3. substr 第 1 个参数为负值时，会与 length 相加 还为负值的时候，为 0 ，然后运算

总结：​

1.截取数组用 slice 2.slice 参数为负值时，会与 length 相加还为负值的话，记为 0 （转化后，第二个参数不大于第一个参数，截取为 ''） 3.substring 参数为负值时候会记为 0，并且会对参数默认进行排序（小的在前） 4.substr 第一个参数为负值时，会与 length 相加 还为负值的话，记为 0 ，然后运算。第二个参数是截取的位数，为负值的时候，截取的是 '' （ ECMAscript 没有对该方法进行标准化，因此反对使用它）
