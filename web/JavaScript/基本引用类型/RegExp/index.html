RegExp

Date 对象用于将文本与一个模式匹配

1. 创建 RegExp 对象：​

有两种方法可以创建一个 RegExp 对象：一种是字面量，另一种是构造函数

  1. 字面量：由斜杠 / 包围而不是引号包围

  2. 构造函数的字符串参数：由引号而不是斜杠包围

;/ab+c/i //字面量形式
new RegExp('ab+c', 'i') // 首个参数为字符串模式的构造函数
new RegExp(/ab+c/, 'i') // 首个参数为常规字面量的构造函数

模式匹配的标识符：

标志描述
g全局搜索
i不区分大小写搜索
m多行搜索
s允许 . 匹配换行符
u使用 unicode 码的模式进行匹配
y执行“粘性 (sticky)”搜索，匹配从目标字符串的当前位置开始

2. 实例属性：​

1. flags：​

含有 RegExp 对象 flags 的字符串

返回一个字符串，由当前正则表达式对象的模式匹配的标识符组成 从左到右，即"gimuy"

2. dotAll：​

. 是否要匹配新行（newlines）

3. global：​

针对字符串中所有可能的匹配项测试正则表达式，还是仅针对第一个匹配项

如果使用了 "g" 标志，则返回 true；否则返回 false

4. ignoreCase：​

匹配文本的时候是否忽略大小写

5. multiline：​

是否进行多行搜索

6. source：​

正则表达式的文本

7. sticky：​

搜索是否是 sticky

8. unicode：​

Unicode 功能是否开启

3. 实例方法：​

1. exec()：​

在该字符串中执行匹配项的搜索

2. test()：​

该正则在字符串里是否有匹配

3. toString()：​

返回表示指定对象的字符串。重写 Object.prototype.toString() 方法
