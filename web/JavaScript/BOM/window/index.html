window

BOM (Browser Object Model) 浏览器对象模型，是浏览器的一个对象模型，它提供了一个统一的接口，让我们可以访问浏览器的各种功能。

BOM 的核心是 window 对象，它是浏览器的全局对象，它是所有其他对象的公共祖先

window 对象在浏览器中有两重身份，一个是 ECMAScript 中的 Global 对象，另一个就是浏览器窗口的 JavaScript 接口

这就意味着网页中定义的所有对象、变量和函数都以 window 作为全局对象，都可以直接访问定义在上面的全局方法和属性

1. Global 作用域：​

通过 var 声明的所有全局变量和函 数都会变成 window 对象的属性和方法

var age = 29
var sayAge = () => alert(this.age)
alert(window.age) // 29
sayAge() // 29
window.sayAge() // 29

JavaScript 中有很多对象都暴露在全局作用域中，比如 location 和 navigator 对象，这些对象都是全局对象，因此他们也是 window 对象的属性

2. 静态属性：​

1. closed 非标准的 只读：​

这个属性指示当前窗口是否关闭

2. console 只读：​

返回 console 对象的引用，该对象提供了对浏览器调试控制台的访问

3. content 只读：​

返回当前 window 的 content 元素的引用

4. controllers 非标准的 只读：​

返回当前 chrome window 的 XUL 控制器对象

5. crypto 只读：​

返回浏览器 crypto 对象

6. devicePixelRatio 非标准的 只读：​

返回当前显示器的物理像素和设备独立像素的比例

7. dialogArguments 只读：​

获取在调用 window.showModalDialog() 时传递给窗口的参数（如果它是一个对话框）。这是一个 nsIArray

8. document 只读：​

返回对当前窗口所包含文档的引用（DOM）

9. frameElement 只读：​

返回嵌入窗口的元素，如果未嵌入窗口，则返回 null

10. frames 只读：​

返回当前窗口中所有子窗体的数组

11. fullScreen：​

此属性表示窗口是否以全屏显示

12. history 只读：​

返回一个对 history 对象的引用

13. innerHeight 只读：​

获得浏览器窗口的内容区域的高度，包含水平滚动条 (如果有的话)

14. innerWidth 只读：​

获得浏览器窗口的内容区域的宽度，包含垂直滚动条 (如果有的话)

15. isSecureContext 实验性的 只读：​

指出上下文环境是否能够使用安全上下文环境的特征

16. length 只读：​

返回窗口中的 frames 数量。参见 window.frames

17. location：​

获取、设置 window 对象的 location，或者当前的 URL

18. locationbar 只读：​

返回 locationbar 对象，其可视性可以在窗口中切换

19. localStorage 只读：​

返回用来存储只能在创建它的源下访问的数据的本地存储对象的引

20. menubar 只读：​

返回菜单条对象，它的可视性可以在窗口中切

21. name：​

获取/设置窗口的名称

22. navigator 只读：​

返回对 navigator 对象的引用

23. opener：​

返回对打开当前窗口的那个窗口的引用

24. outerHeight 只读：​

返回浏览器窗口的外部高度

25. outerWidth 只读：​

返回浏览器窗口的外部宽度

26. pageXOffset 只读：​

window.scrollX 的别名

27. pageYOffset 只读：​

window.scrollY 的别名

28. parent 只读：​

返回当前窗口或子窗口的父窗口的引用

29. personalbar 只读：​

返回 personalbar 对象，它的可视性可以在窗口中切换

30. screen 只读：​

返回对与窗口关联的屏幕对象的引用

31. screenX 和 screenLeft 只读：​

这两个属性都返回从用户浏览器视口的左边框到屏幕左侧的水平距离

32. screenY 和 screenTop 只读：​

这两个属性都返回从用户浏览器视口的顶部边框到屏幕顶部的垂直距离

33. scrollbars 只读存：​

返回滚动条对象，其可见性可以在窗口中切换

34. scrollMaxX 非标准的 只读：​

窗口可以水平滚动到的最大偏移量，即文档宽度减去视口宽度

35. scrollMaxY 非标准的 只读：​

窗口可以垂直滚动到的最大偏移量（即文档高度减去视口高度）

36. scrollX 只读：​

返回文档已经水平滚动的像素数

37. scrollY 只读：​

返回文档已经垂直滚动的像素数

38. self 只读：​

返回对窗口对象本身的对象引用

39. sessionStorage：​

返回对会话存储对象的引用，该对象用于存储只能由创建它的源访问的数据

40. sidebar 非标准的 只读：​

返回对侧边栏窗口对象的引用

41. speechSynthesis 只读：​

返回一个对象，它是使用 Web Speech API 语音合成功能 SpeechSynthesis 的入口点

42. status：​

获取/设置浏览器底部状态栏中的文本

43. statusbar 只读：​

返回状态栏对象，其可见性可以在窗口中切换

44. toolbar 只读：​

返回工具栏对象，其可见性可以在窗口中切换

45. top 只读：​

返回对窗口层次结构中最顶层窗口的引用。该属性是只读的

46. visualViewport 只读：​

返回一个 VisualViewport 对象，该对象表示给定窗口的可视视口

47. window 只读：​

返回对当前窗口的引用

3. 静态方法：​

1. alert()：​

显示一个警告对话框，上面显示有指定的文本内容以及一个"确定"按钮

2. blur()：​

将焦点移出顶层窗口

3. cancelAnimationFrame()：​

取消一个先前通过调用 window.requestAnimationFrame() 方法添加到计划中的动画帧请求

4. cancelIdleCallback()：​

用于取消之前调用 window.requestIdleCallback() 的回调

5. clearImmediate()：​

此方法用来清除 window.setImmediate()

6. close()：​

关闭当前窗口或某个指定的窗口，该方法只能由 Window.open() 方法打开的窗口的 window 对象来调用

7. confirm()：​

显示一个具有一个可选消息和两个按钮（确定和取消）的模态对话框

8. find()：​

在一个页面中搜索指定的字符串

9. focus()：​

请求将窗口显示在前（靠近屏幕），这可能由于用户设置而失败，并且该窗口在方法返回之前不能保证会显示在最前

10. getComputedStyle()：​

返回一个对象，该对象在应用活动样式表并解析这些值可能包含的任何基本计算后报告元素的所有 CSS 属性的值

11. getSelection()：​

返回一个 Selection 对象，表示用户选择的文本范围或光标的当前位置

12. matchMedia()：​

返回一个新的 MediaQueryList 对象，表示指定的 媒体查询 字符串解析后的结果。返回的 MediaQueryList 可被用于判定 Document 是否匹配媒体查询，或者监控一个 document 来判定它匹配了或者停止匹配了此媒体查询

13. minimize()：​

让当前浏览器窗口最小化（可以通过调用 window.moveTo() 方法让窗口恢复正常显示）

14. moveBy()：​

根据指定的值，移动当前窗口

15. moveTo()：​

将当前窗口移动到指定的坐标位置

16. open()：​

打开一个新窗口

17. postMessage()：​

为一个窗口向另一个窗口发送数据字符串提供了一种安全方法，该窗口不必与第一个窗口处于相同的域中

18. print()：​

打开打印对话框以打印当前文档

19. prompt()：​

返回用户在提示对话框中输入的文本

20. requestAnimationFrame()：​

告诉浏览器——你希望执行一个动画，并且要求浏览器在下次重绘之前调用指定的回调函数更新动画。该方法需要传入一个回调函数作为参数，该回调函数会在浏览器下一次重绘之前执行

21. requestIdleCallback()：​

插入一个函数，这个函数将在浏览器空闲时期被调用。这使开发者能够在主事件循环上执行后台和低优先级工作，而不会影响延迟关键事件，如动画和输入响应

22. resizeBy()：​

将当前窗口调整到一定的大小

23. resizeTo()：​

动态调整窗口的大小

24. scroll()：​

滚动窗口到文档中的特定位置

25. scrollBy()：​

在窗口中按指定的偏移量滚动文档

26. scrollByLines()：​

按给定行数滚动文档

27. scrollByPages()：​

按指定页数滚动当前文档

28. scrollTo()：​

滚动到文档中的特定坐标集

29. setCursor()：​

更改当前窗口的光标

30. sizeToContent()：​

根据内容设置窗口大小

31. stop()：​

这个方法停止窗口加载

32. updateCommands()：​

更新当前 chrome 窗口 (UI) 命令的状态

4. 特殊方法：​

1. addEventListener()：​

将指定的监听器注册到 EventTarget 上，当该对象触发指定的事件时，指定的回调函数就会被执行。事件目标可以是一个文档上的元素 Element, Document 和 Window 或者任何其他支持事件的对象 (比如 XMLHttpRequest)

2. atob()：​

对经过 base-64 编码的字符串进行解码

3. btoa()：​

从 String 对象中创建一个 base-64 编码的 ASCII 字符串，其中字符串中的每个字符都被视为一个二进制数据字

4. clearInterval()：​

可取消先前通过 setInterval() 设置的重复定时任务

5. clearTimeout()：​

取消了先前通过调用 setTimeout()建立的定时器

6. createImageBitmap()：​

它接受各种不同的图像来源，并返回一个 Promise, resolve 为 ImageBitmap. 可选的参数，图像被剪裁成自（sx，sy）且宽度为 sw，高度为 sh 的像素的矩形

7. fetch()：​

用于发起获取资源的请求。它返回一个 promise，这个 promise 会在请求响应后被 resolve，并传回 Response 对象

8. removeEventListener()：​

删除使用 addEventListener() 方法添加的事件

9. setInterval()：​

重复调用一个函数或执行一个代码片段，在每次调用之间具有固定的时间间隔（毫秒数）

10. setTimeout()：​

设置一个定时器，该定时器在定时器到期后执行一个函数或指定的一段代码（毫秒数）
