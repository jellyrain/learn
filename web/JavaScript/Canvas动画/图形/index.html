图像

1. 绘制矩形：

1. fillRect(x, y, width, height)：​

绘制一个填充了内容的矩形，这个矩形的开始点（左上点）在(x, y) ，它的宽度和高度分别由 width 和 height 确定，填充样式由当前的 fillStyle 决定

2. strokeRect(x, y, width, height)：​

绘制一个描边矩形，其起点为(x, y) ，其大小由宽度和高度指定

3. clearRect(x, y, width, height)：​

在一个矩形区域内设置所有像素都是透明的 (rgba(0,0,0,0))。这个矩形范围的左上角在 (x, y)，宽度和高度分别由 width 和 height 确定

2. 绘制路径：

图形的基本元素是路径。路径是通过不同颜色和宽度的线段或曲线相连形成的不同形状的点的集合。一个路径，甚至一个子路径，都是闭合的。使用路径绘制图形需要一些额外的步骤

  1. 首先，你需要创建路径起始点
  2. 然后你使用画图命令去画出路径
  3. 之后你把路径封闭
  4. 一旦路径生成，你就能通过描边或填充路径区域来渲染图

绘制一个三角形：

ctx.beginPath()
ctx.moveTo(75, 50)
ctx.lineTo(100, 75)
ctx.lineTo(100, 25)
ctx.fill()

1. 路径基本函数：​

1. beginPath()：​

新建一条路径，生成之后，图形绘制命令被指向到路径上生成路径（分隔上下路径）

2. closePath()：​

闭合路径之后图形绘制命令又重新指向到上下文中（自动闭合路径）

3. stroke()：​

通过线条来绘制图形轮廓

4. fill()：​

通过填充路径的内容区域生成实心的图形

2. 线：​

1. moveTo(x, y)：​

将笔触移动到指定的坐标 x 以及 y 上 通常会使用 moveTo() 函数设置起点。我们也能够使用 moveTo() 绘制一些不连续的路径

2. lineTo(x, y)：​

绘制一条从当前位置到指定 x 以及 y 位置的直线

3. 矩形：​

1. rect(x, y, width, height)：​

绘制一个左上角坐标为（x,y），宽高为 width 以及 height 的矩形

4. 圆弧：​

1. arc(x, y, radius, startAngle, endAngle, anticlockwise)：​

画一个以（x,y）为圆心的以 radius 为半径的圆弧（圆），从 startAngle 开始到 endAngle 结束，按照 anticlockwise（Boolean） 给定的方向（默认为顺时针）来生成

2. arcTo(x1, y1, x2, y2, radius)：​

根据给定的控制点和半径画一段圆弧，再以直线连接两个控制点

5. 曲线：​

1. quadraticCurveTo(cpx, cpy, x, y)：​

绘制一条二次贝塞尔曲线，控制点为 cpx, cpy，终点为 x, y

2. bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)：​

绘制一条三次贝塞尔曲线，控制点 1 为 cp1x, cp1y，控制点 2 为 cp2x, cp2y，终点为 x, y

3. 样式与颜色：

1. 色彩：​

1. fillStyle：​

设置填充色，可以是颜色名，颜色字符串，颜色数组，或者颜色对象

ctx.fillStyle = 'red'
ctx.fillStyle = '#f00'
ctx.fillStyle = [255, 0, 0]
ctx.fillStyle = { r: 255, g: 0, b: 0 }

2. strokeStyle：​

设置描边色，可以是颜色名，颜色字符串，颜色数组，或者颜色对象

ctx.strokeStyle = 'red'
ctx.strokeStyle = '#f00'
ctx.strokeStyle = [255, 0, 0]
ctx.strokeStyle = { r: 255, g: 0, b: 0 }

2. 透明度：​

1. globalAlpha：​

设置全局透明度，取值范围 0-1

ctx.globalAlpha = 0.5

3. 线型：​

1. lineWidth：​

设置线条宽度

ctx.lineWidth = 10

2. lineCap：​

设置线条的结束端点样式，可以是 butt, round, square

ctx.lineCap = 'round'

3. lineJoin：​

设置线条的结束端点样式，可以是 miter, round, bevel

ctx.lineJoin = 'round'

4. miterLimit：​

设置斜接面限制比例的属性。当获取属性值时，会返回当前的值（默认值是 10.0 ）

ctx.miterLimit = 10

5. 虚线：​

1. setLineDash(segments)：​

设置虚线模式，可以是一个数组，或者一个数组的数组，或者一个空数组，表示不使用虚线

ctx.setLineDash([5, 10])

2. getLineDash()：​

获取虚线模式

ctx.getLineDash()

3. lineDashOffset：​

设置虚线偏移量

ctx.lineDashOffset = 10

4. 渐变：​

1. createLinearGradient(x0, y0, x1, y1)：​

创建一个线性渐变，x0, y0, x1, y1 是渐变的起点和终点

ctx.createLinearGradient(0, 0, 100, 100)

2. createRadialGradient(x0, y0, r0, x1, y1, r1)：​

创建一个径向渐变，x0, y0, r0, x1, y1, r1 是渐变的起点和终点，r0, r1 是渐变的半径

ctx.createRadialGradient(0, 0, 10, 0, 0, 100)

3. CanvasGradient.addColorStop(offset, color)：​

添加一个渐变点，offset 是渐变点的位置，color 是渐变点的颜色

gradient.addColorStop(0, 'red')
gradient.addColorStop(1, 'blue')

5. 图案样式：​

1. createPattern(image, repetition)：​

创建一个图案，image 是图案的图像，repetition 是图案的重复方式，可以是 repeat, repeat-x, repeat-y, no-repeat

ctx.createPattern(image, 'repeat')

6. 阴影：​

1. shadowColor：​

设置阴影颜色，可以是颜色名，颜色字符串，颜色数组，或者颜色对象

ctx.shadowColor = 'red'

2. shadowBlur：​

设置阴影模糊级别，取值范围 0-100

ctx.shadowBlur = 10

3. shadowOffsetX：​

设置阴影水平偏移，取值范围 -1000-1000

ctx.shadowOffsetX = 10

4. shadowOffsetY：​

设置阴影垂直偏移，取值范围 -1000-1000

ctx.shadowOffsetY = 10
