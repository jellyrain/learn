变换

transform：​

允许你旋转，缩放，倾斜或平移给定元素

只能转换由盒模型定位的元素

值描述
none不应用任何变换
<transform-function>要应用的一个或多个 CSS 变换函数。变换函数按从左到右的顺序相乘，这意味着复合变换按从右到左的顺序有效地应用

translate()：​

用于移动元素在平面上的位置。这种变换的特点是矢量的坐标定义了它在每个方向上的移动量

translateX() 和 translateY() 的缩写

transform: translate(tx);

transform: translate(tx, ty);
值描述
tx表示要移动矢量的横坐标
ty表示要移动矢量的纵坐标。如果不写则默认为零

translate3d()：​

用于移动元素在 3D 空间中的位置。这种变换的特点是三维矢量的坐标定义了它在每个方向上的移动量

translateX() 和 translateY() 和 translateZ() 的缩写

transform: translate3d(tx, ty, tz);
值描述
tx表示要移动矢量的横坐标
ty表示要移动矢量的纵坐标
tz代表移动矢量的 z 轴坐标 (该值不能使用百分比)

scale()：​

可改变元素的大小(缩放)

scaleX() 和 scaleY() 的缩写

transform: scale(sx);

transform: scale(sx, sy);
值描述
sx代表缩放矢量的横坐标
sy代表缩放矢量的纵坐标。如果不存在，则其默认值为 sx，从而导致保持元素形状进行均匀缩放

scale3d()：​

可改变元素的大小(缩放)

scaleX() 和 scaleY() 和 scaleZ() 的缩写

transform: scale(sx, sy, sz);
值描述
sx代表缩放矢量的横坐标
sy代表缩放矢量的纵坐标
sz代表缩放矢量的 z 轴坐标

rotate()：​

定义一个旋转属性

将元素在不变形的情况下旋转到 不动点 周围 (如 transform-origin 属性所指定)

transform: rotate(a);
值描述
a该参数表示 <angle> 代表旋转的角度。正角表示顺时针旋转，负角表示逆时针旋转

rotate3d()：​

定义一个 3D 旋转功能，该旋转使元素能够绕固定轴移动而不变形

transform: rotate3d(x, y, z, a);
值描述
x描述旋转轴向量的 x 坐标
y描述旋转轴向量的 y 坐标
z描述旋转轴向量的 z 坐标
a该参数为 <angle> 代表旋转的角度。正角度表示顺时针旋转，负角度表示逆时针旋转

rotateX()：​

将元素在横坐标上旋转而不使其变形的方法。其运动的程度由指定的角度来定义；如果是正的，则为顺时针旋转，如果是负的，则是逆时针旋转

旋转所经过的原点，由 transform-origin CSS 属性定义

transform: rotateX(a);
值描述
a该参数为 <angle> 代表旋转的角度。正角度表示顺时针旋转，负角度表示逆时针旋转

rotateY()：​

将元素在纵坐标上旋转而不使其变形的方法。其运动的程度由指定的角度来定义；如果是正的，则为顺时针旋转，如果是负的，则是逆时针旋转

旋转所经过的原点，由 transform-origin CSS 属性定义

transform: rotateY(a);
值描述
a该参数为 <angle> 代表旋转的角度。正角度表示顺时针旋转，负角度表示逆时针旋转

rotateZ()：​

定义了将元素在 z 轴上旋转而不使其变形的方法。其运动的程度由指定的角度来定义；如果是正的，则为顺时针旋转，如果是负的，则是逆时针旋转

旋转所经过的原点，由 transform-origin CSS 属性定义

transform: rotateZ(a);
值描述
a该参数为 <angle> 代表旋转的角度。正角度表示顺时针旋转，负角度表示逆时针旋转

perspective()：​

定义了 z = 0 平面与用户之间的距离，以便给三维定位元素一定透视度。当每个 3D 元素的 z > 0 时会显得比较大，而在 z < 0 时会显得比较小。其影响的程度由这个属性的值来决定

transform: perspective(l);
值描述
l该参数是一个 <length> 给定从用户（显示屏）到 z = 0 平面的距离。它用于将透视图转换应用于元素。如果它是 0 或负值，则不应用透视变换

skew()：​

是一种用于拉伸，或者说是平移，该函数会使得在每个方向上扭曲元素上的每个点以一定角度。这是通过将每个坐标增加一个与指定角度成比例的值和到原点的距离来完成的。离原点越远，拉伸的值就越大

skewX() 和 skewY() 的缩写

transform: skew(ax);

transform: skew(ax, ay);
值描述
ax表示用于沿着横坐标扭曲元素的角度
ay表示用于沿纵坐标扭曲元素的角度

matrix()：​

用六个指定的值来指定一个均匀的二维（2D）变换矩阵。这个矩形中的常量值是不作为参数进行传递的，其他的参数则在主要列的顺序中描述

transform: matrix(a, b, c, d, tx, ty);
值描述
a b c d以 <number> 的格式来描述线性变换
tx ty以 <number> 的格式来描述变换的量

matrix3d()：​

用一个 4 × 4 的齐次矩阵来描述一个三维（3D）变换。16 个参数都在主要列的顺序中描述

transform: matrix3d(
  a1,
  b1,
  c1,
  d1,
  a2,
  b2,
  c2,
  d2,
  a3,
  b3,
  c3,
  d3,
  a4,
  b4,
  c4,
  d4
);
值描述
a1 b1 c1 d1 a2 b2 c2 d2 a3 b3 c3 d3 d4以 <number> 的格式来描述线性变换
a4 b4 c4以 <number> 的格式来描述变换的量

transform-origin：​

更改一个元素变形的原点

/* One-value syntax */
transform-origin: 2px;
transform-origin: bottom;

/* x-offset | y-offset */
transform-origin: 3cm 2px;

/* x-offset-keyword | y-offset */
transform-origin: left 2px;

/* x-offset-keyword | y-offset-keyword */
transform-origin: right top;

/* y-offset-keyword | x-offset-keyword */
transform-origin: top right;

/* x-offset | y-offset | z-offset */
transform-origin: 2px 30% 10px;

/* x-offset-keyword | y-offset | z-offset */
transform-origin: left 5px -3px;

/* x-offset-keyword | y-offset-keyword | z-offset */
transform-origin: right bottom 2cm;

/* y-offset-keyword | x-offset-keyword | z-offset */
transform-origin: bottom right 2cm;
keywordvalue
keywordvalue
left0%
center50%
right100%
top0%
bottom100%

transform-box：​

定义了与 transform、transform-origin 这两个属性有关联的布局框

值描述
border-box使用边框作为参考框。表的参考框是包裹着该表的边框，而不是其表框
fill-box使用对象边界框作为参考框
view-box使用最近的 SVG 视口作为参考框

transform-style：​

设置元素的子元素是位于 3D 空间中还是平面中

值描述
flat设置元素的子元素位于该元素的平面中
preserve-3d指示元素的子元素应位于 3D 空间中
