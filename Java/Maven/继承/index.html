继承

Maven中的继承是针对 Maven工程之间的继承，父工程可以统一管理一些事物，子工程就可以继承的方式获取父工程的资源

1. 原始版：​

  1. 创建父工程：

    <groupId>com.jellyrain.maven</groupId>
    <artifactId>Parent</artifactId>
    <!-- 父工程的打包方式为pom（pom：不打包） -->
    <packaging>pom</packaging>
    <version>1.0-SNAPSHOT</version>

  2. 在子工程中引用父工程：

    <!--继承-->
    <parent>
        <groupId>com.atguigu.maven</groupId>
        <artifactId>Parent</artifactId>
        <version>1.0-SNAPSHOT</version>
    <!--指定从当前pom.xml文件出发寻找父工程的pom.xml文件的相对路径-->
    <relativePath>../Parent/pom.xml</relativePath>
    </parent>

    此时如果子工程的 groupId 和 version 如果和父工程重复则可以删除

  3. 父工程管理依赖版本：

    <!--依赖管理-->
    <dependencyManagement>
        <dependencies>
            <dependency>
                <groupId>junit</groupId>
                <artifactId>junit</artifactId>
                <version>4.0</version>
                <scope>test</scope>
            </dependency>
        </dependencies>
    </dependencyManagement>

    在子项目中重新指定需要的依赖，删除范围和版本号

    <dependency>
        <groupId>junit</groupId>
        <artifactId>junit</artifactId>
    </dependency>

2. IDEA版：​

选中父工程 使用 module 创建子工程

自动 继承

父工程 自动变成 pom

只需要管理依赖版本即可，src 也可以删除
