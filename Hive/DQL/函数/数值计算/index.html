数值计算

1. 近似函数：round​

语法：round(double a)

返回类型：bigint

说明：返回 double 类型的整数值部分 （遵循四舍五入）

select round(3.1415926);

2. 指定精度近似函数：round​

语法：round(double a)

返回类型：double

说明：返回指定精度 d 的 double 类型

select round(3.1415926, 4);

3. 向下取整函数：floor​

语法：floor(double a)

返回类型：double

说明：返回不大于 a 的最大整数

select floor(3.1415926, 4);

4. 向上取整函数：ceil​

语法：ceil(double a)

返回类型：bigint

说明：返回不小于 a 的最小整数

select ceil(3.1415926, 4);

5. 向上取整函数：ceiling​

语法：ceiling(double a)

返回类型：bigint

说明：与 ceil 功能相同

select ceiling(3.1415926, 4);

6. 取随机函数：rand​

语法：rand()，rand(int seed)

返回类型：bigint

说明：返回一个 0 到 1 范围内的随机数。如果指定种子 seed，则会得到一个稳定的随机数序列。

select rand();

select rand(100);

7. 自然指数函数：exp​

语法：exp(double a)

返回类型：double

说明：返回自然对数 e 的 a 次方

select exp(2);

8. 以 10 为底的对数函数：log10​

语法：log10(double a)

返回类型：double

说明：返回以 10 为底的 a 的对数

select log10(100);

9. 以 2 为底的对数函数：log2​

语法：log2(double a)

返回类型：double

说明：返回以 2 为底的 a 的对数

select log2(8);

10. 对数函数：log​

语法：log2(double base, double a)

返回类型：double

说明：返回以 base 为底的 a 的对数

select log(2, 1024);

11. 幂运算函数：pow​

语法：pow(double a, double p)

返回类型：double

说明：返回以 a 的 p 次幂

select pow(2, 5);

12. 幂运算函数：power​

语法：power(double a, double p)

返回类型：double

说明：返回以 a 的 p 次幂，与 pow 功能相同。

select power(2, 10);

13. 平方根函数：sqrt​

语法：sqrt(double a)

返回类型：double

说明：返回 a 的 平方根

select sqrt(100);

14. 二进制函数：bin​

语法：bin(bigint a)

返回类型：string

说明：返回 a 的 二进制代码表示

select bin(7);

15. 十六进制函数：hex​

语法：hex(bigint a)

返回类型：string

说明：如果变量是 int 类型，那么返回 a 的十六进制表示；如果变量是 string 类型，则返回该字符串的十六进制表示。

select hex(17);

select hex('abc');

16. 反转十六进制函数：unhex​

语法：unhex(string a)

返回类型：string

说明：返回该十六进制数或字符串所反转的字符串

select unhex(616263);

select unhex('616263');

17. 进制转换函数：conv​

语法：conv(bigint num, int from_base, int to_base)

返回类型：string

说明：将数值 num 从 from_base 进制转化到 to_base 进制

select conv(17, 10, 2);

select conv(17, 10, 16);

18. 绝对值函数：abs​

语法：abs(int a )，absdouble a)

返回类型：int double

说明：返回数值 a 的绝对值

select abs(-1);

select abs(3.14);

19. 正取余函数：pmod​

语法：pmod(int a, int b)，pmod(double a, double b)

返回类型：int double

说明：返回正的 a 除以 b 的余数

select pmod(9, 4);

select pmod(-9, 4);

20. 正弦函数：sin​

语法：sin(double a)

返回类型：double

说明：返回 a 的正弦值

select sin(0.8);

21. 反正弦函数：asin​

语法：asin(double a)

返回类型：double

说明：返回 a 的反正弦值

select sin(0.7173560908995228);

22. 余弦函数：cos​

语法：cos(double a)

返回类型：double

说明：返回 a 的余弦值

select cos(0.8);

23. 反余弦函数：acos​

语法：acos(double a)

返回类型：double

说明：返回 a 的反余弦值

select acos(0.6967067093471654);

24. positive 函数：positive​

语法：positive(int a)，positive(double a)

返回类型：int double

说明：返回 a 自身

select positive(-1);

select positive(3.14);

25. negative 函数：negative​

语法：negative(int a)，negative(double a)

返回类型：int double

说明：返回 a 的相反数

select negative(-1);

select negative(3.14);
