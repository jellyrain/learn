where

1. where基础使用：​

如需有条件的从表中选取数据，可将 where子句添加到 select 语句。

SELECT 列名称 FROM 表名称 WHERE 列名称 运算符 值

2. 操作符：​

1. 基础操作符：​

操作符描述
=等于
<> 或 !=不等于
>大于
<小于
>=大于等于
<=小于等于

2.特殊操作符：​

1. and：​

两个条件做 与 运算。

select * from 表 where 判断条件1 and 判断条件2
2. or：​

两个条件做 或 运算。

select * from 表 where 判断条件1 and 判断条件2
3. not：​

条件取反 非 运算。

select * from 表 where not 判断条件
4. any：​

只要满足内部任意一个就可以通过，需要使用基础操作符。

select * from 表 where 姓名 =any('小明', '小红')
  1. any后面可以接子查询
  2. >any：表示大于最小值
  3. <any：表示小于最大值
  4. 可以在any 前加 not 取反
5. all：​

只要满足内部任意一个就可以通过，需要使用基础操作符。

select * from 表 where 工资 > all(100, 200)
  1. all后面可以接子查询
  2. >all: 表示大于最大值
  3. <all：表示小于最小值
  4. 可以在all 前加 not 取反
6. in：​

逻辑上等于：=any

select * from 表 where 姓名 in('小明', '小红')

  1. all后面可以接子查询

  2. 可以在in 前加 not 取反

  3. 如果是多列子查询，需要in前面也是多列

select * from 表 where (姓名, 工资) in(select 姓名, 工资 from 表)
7. exists：​

逻辑上等于：=any

select * from 表1 where exists(select * from 表2 where 表1.姓名 = 表2.姓名)
  1. 可以在 exists 前加 not 取反
8. between：​

来显示在一个区间内的值。 逻辑上等于：值 => xx and 值 <= xx

select * from 表 where 工资 between 1000 and 3000
等价
select * from 表 where 工资 => 1000 and 工资 <= 3000
9. like：​

用来模糊查询字符串使用。

  1. 特殊字符：

    字符描述
    %代表零个或多个字符(任意个字符)
    _代表一个字符
    [charlist]字符列中的任何单一字符
    [^charlist]或者[!charlist]不在字符

  2. 转义字符：

    需要使用特殊符号的时候需要先进行转义 escape 设置转义字符(一般使用 \ 反斜杠)

select * from 表 where 姓名 like '%红%'

select * from 表 where 姓名 like '\%' escape '\'
10.rlike：​

rlike子句是 Hive 中这个功能的一个扩展，其可以通过Java的正则表达式这个更强大的语言来指定匹配条件（扩展like）

11. is (not) null：​

判断是否是空值

select * from 表 where 姓名 is null
