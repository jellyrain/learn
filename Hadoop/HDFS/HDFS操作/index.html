1. 上传指令：​

[-moveFromLocal <localsrc> ... <dst>] # 从本地剪切粘贴到HDFS
[-copyFromLocal [-f] [-p] <localsrc> ... <dst>] # 从本地文件系统中拷贝文件到HDFS路径去
[-put [-f] [-p] <localsrc> ... <dst>] # 等同于 copyFromLocal，生产环境更习惯用put
[-appendToFile <localsrc> ... <dst>] # 追加一个文件到已经存在的文件末尾

2. 下载指令：​

[-moveToLocal <src> <localdst>]  # 从 HDFS 剪切粘贴到 本地
[-copyToLocal [-p] [-ignoreCrc] [-crc] <src> ... <localdst>] # 从 HDFS 拷贝到本地
[-get [-p] [-ignoreCrc] [-crc] <src> ... <localdst>] # 等同于 copyToLocal，生产环境更习惯用get

3. 直接操作 HDFS：​

[-help [cmd ...]] # 查看指令帮助

[-ls [-d] [-h] [-R] [<path> ...]] # 显示目录信息
[-cat [-ignoreCrc] <src> ...] # 显示文件内容

# Linux 文件系统中的用法一样，修改文件所属权限
[-chgrp [-R] GROUP PATH...]
[-chmod [-R] <MODE[,MODE]... | OCTALMODE> PATH...]
[-chown [-R] [OWNER][:[GROUP]] PATH...]

[-mkdir [-p] <path> ...] # 创建路径
[-cp [-f] [-p] <src> ... <dst>] # 从 HDFS 一个路径拷贝到HDFS的另一个路径
[-mv <src> ... <dst>] # 在 HDFS 目录中移动文件
[-tail [-f] [-s <sleep interval>] <file>] # 显示一个文件的末尾1kb的数据
[-rm [-f] [-r|-R] [-skipTrash] <src> ...] # 删除目录或目录里面内容
[-du [-s] [-h] <path> ...] # 统计文件夹的大小信息
[-setrep [-R] [-w] <rep> <path> ...] # 设置HDFS中文件的副本数量

这里设置的副本数只是记录在NameNode的元数据中，是否真的会有这么多副本，还得看DataNode的数量。因为目前只有3台设备，最多也就3个副本，只有节点数的增加到10台时，副本数才能达到10

2. HDFS 的 API 操作：​

1. 环境准备：​

  1. 配置 HADOOP_HOME 环境变量：你安装的地址

  2. 配置 Path 环境变量：%HADOOP_HOME%\bin

    验证Hadoop环境变量是否正常。双击winutils.exe

    如果报错：安装微软运行库（不提供）

  3. 创建 Maven 工程，添加如下依赖包

<dependencies>
    <dependency>
        <groupId>org.apache.hadoop</groupId>
        <artifactId>hadoop-client</artifactId>
        <version>3.1.3</version>
    </dependency>
    <dependency>
        <groupId>junit</groupId>
        <artifactId>junit</artifactId>
        <version>4.12</version>
    </dependency>
    <dependency>
        <groupId>org.slf4j</groupId>
        <artifactId>slf4j-log4j12</artifactId>
        <version>1.7.30</version>
    </dependency>
</dependencies>

2. 获取文件对象：​

// 1 获取文件系统
Configuration configuration = new Configuration();

// FileSystem fs = FileSystem.get(new URI("hdfs://hadoop102:8020"), configuration); jellyrain 是 用户
FileSystem fs = FileSystem.get(new URI("hdfs://hadoop102:8020"), configuration,"jellyrain");

// 2 创建目录
fs.mkdirs(new Path("/xiyou/huaguoshan/"));

// 3 关闭资源
fs.close();

3. HDFS API 操作：​

注意：以下简写，也就是说 1 3 步骤省略 只提供 2 核心逻辑代码

配置文件参数优先级排序（优先级 由上到下）：

（1）客户端代码中设置的值

（2）ClassPath（项目）下的用户自定义配置文件

（3）然后是服务器的自定义配置（xxx-site.xml）

（4）服务器的默认配置（xxx-default.xml）

1. 文件上传：​

fs.copyFromLocalFile(false, true, new Path("d:/sunwukong.txt"), new Path("/xiyou/huaguoshan"));

参数描述：

  1. 是否上传后删除本地原文件
  2. 是否允许覆盖
  3. 本地原文件路径
  4. 上传位置路径

2. 文件下载：​

fs.copyToLocalFile(false, new Path("/xiyou/huaguoshan/sunwukong.txt"), new Path("d:/sunwukong2.txt"), true);

参数描述：

  1. 是否删除原文件
  2. 原文件路径
  3. 下载到本地的路径
  4. 是否不开启文件校验

3. 文件删除：​

fs.delete(new Path("/xiyou"), true);

参数描述：

  1. 要删除的路径
  2. 是否开启递归删除

4. 文件更名和移动：​

fs.rename(new Path("/xiyou/huaguoshan/sunwukong.txt"), new Path("/xiyou/huaguoshan/meihouwang.txt"));

参数描述：

  1. 原文件路径
  2. 目标文件目录

5. 文件详情查看：​

// 返回的是迭代器
RemoteIterator<LocatedFileStatus> listFiles = fs.listFiles(new Path("/"), true);

while (listFiles.hasNext()) {
    LocatedFileStatus fileStatus = listFiles.next();

    System.out.println("========" + fileStatus.getPath() + "=========");
    System.out.println(fileStatus.getPermission());
    System.out.println(fileStatus.getOwner());
    System.out.println(fileStatus.getGroup());
    System.out.println(fileStatus.getLen());
    System.out.println(fileStatus.getModificationTime());
    System.out.println(fileStatus.getReplication());
    System.out.println(fileStatus.getBlockSize());
    System.out.println(fileStatus.getPath().getName());

    // 获取块信息
    BlockLocation[] blockLocations = fileStatus.getBlockLocations();
    System.out.println(Arrays.toString(blockLocations));
}

参数描述：

  1. 原文件路径
  2. 是否开启递归

6. 文件和文件夹判断：​

// 返回的是迭代器
FileStatus[] listStatus = fs.listStatus(new Path("/"));

for (FileStatus fileStatus : listStatus) {
    // 如果是文件
    if (fileStatus.isFile()) {
        System.out.println("f:"+fileStatus.getPath().getName());
    }else {
        System.out.println("d:"+fileStatus.getPath().getName());
    }
}

参数描述：

  1. 原文件路径
