percent_rank

描述：​

percent_rank 函数用来查询指定数值在指定分组序列中的相对位置，返回值是介于 0 和 1 之间的小数值

类型：​

Funciton

格式：​

percent_rank(expr[, expr...]) within group(order by expr2[, expr2...])

返回值：​

number

参数：​

  1. expr[, expr...]（必填）一列或者多列
  2. expr2[, expr2...]（必填）一列或者多列

注意：​

  1. 两个参数的列数量必须一样
  2. 范围：0 <= percent_rank() <= 1
