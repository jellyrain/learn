select

1. select基础使用：​

select 语句 用于从表中选取数据展示。

    select 列名称 from 表名称 (单列查询)
或者：
    select * from 表名称 (全列查询)
或者：
    select LastName, FirstName from Persons (查询名为 "LastName" 和 "FirstName"的列的内容)

2. distinct 去重：​

在表中，可能会包含重复值。这并不成问题，不过，有时您也许希望仅仅列出不同（distinct）的值。

select distinct 列名称 from 表名称

3. 运算：​

select 语句展示的列可以进行运算在显示（数字和日期）。

操作符描述
+加
-减
*乘
/除

操作符优先级：

  1. 乘除的优先级高于加减。
  2. 同一优先级运算符从左向右执行。
  3. 括号内的运算先执行。
select 工资 + 津贴 from 表名称

4. 允许值：​

  1. 字段（列名）
  2. 字符串常量
  3. 数字
  4. null
  5. 特殊关键字：如 sysdate 查询当前时间
  6. 函数返回结果
  7. || 字符串拼装结果
  8. rowid 伪列：插入时生成的物理id
  9. rownum 伪列：查询表时生成的id

5. 列的别名：​

可以重命名一个列的名字，使用 as 关键字 （as 可以省略不写）。

完整：select 工资 as 小钱钱 from 表名称

简写：select 工资 小钱钱 from 表名称

列的别名如果出现 特殊字符 需要使用双引号包裹。

select 工资 "小钱钱 ~(o v o)~" from 表名称
