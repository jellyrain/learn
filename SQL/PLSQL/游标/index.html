游标

可以控制上下文区和处理语句时上下文区会发生些什么事情，游标仅仅是在查询的时候有效的，处理多条数据时使用，一般配合循环语句使用

1. 显式游标：​

1. 第一种写法（loop）：
declare
    声明变量 | 创建记录类型实例
    
    创建游标（如果设置参数（带参游标）就可以在 where，having 中使用参数来判断）：
    cursor 游标名[(参数 类型[, 参数 类型 ....])] is SQL语句;
begin
    打开游标：
    open 游标名[(参数[, 参数 ....])];
    
    loop
        移动一次游标，并且将此行数据赋值给变量：
        fetch 游标名 into 记录类型实例 | 变量名[, 变量名 .....];
        
        循环执行语句;
        
        判断是否无记录，退出循环：
        exit when 游标名%notfound;
    end loop
    
    关闭游标：
    close 游标名
end;

2. 第一种写法（while）：
declare
    声明变量 | 创建记录类型实例
    
    创建游标（如果设置参数（带参游标）就可以在 where，having 中使用参数来判断）：
    cursor 游标名[(参数 类型[, 参数 类型 ....])] is SQL语句;
begin
    打开游标：
    open 游标名[(参数[, 参数 ....])];
    
    fetch 游标名 into 记录类型实例 | 变量名[, 变量名 .....]; 必须先 fetch 一次
    
    while 游标名%found loop
        移动一次游标，并且将此行数据赋值给变量：
        fetch 游标名 into 记录类型实例 | 变量名[, 变量名 .....];
        
        循环执行语句;
        
        判断是否无记录，退出循环：
        exit when 游标名%notfound;
    end loop
    
    关闭游标：
    close 游标名
end;

3. 第二种写法（for）：
declare
    创建游标（如果设置参数（带参游标）就可以在 where，having 中使用参数来判断）：
    cursor 游标名[(参数 类型[, 参数 类型 ....])] is SQL语句;
begin
    for 循环计数器 in 游标名[(参数 类型[, 参数 类型 ....])] loop
        
        循环执行语句;
        
    end loop
end;

4. current of 语句：
通过where current of 语句进行数据更新或删除 就可以直接修改最后取出的数据行 而不用再写 判断条件了

declare
    cursor 游标名 is sql语句;--通过游标更新数据 声明游标必须带 for update
begin
    for ser_emp in 游标名 loop
        delete from emp where current of 游标名;--使用current of 删除数据
    end loop;
end;

注意：

  1. 推荐使用 for 写法 自动开关游标 和 检测是否无记录
  2. 循环计数器 就是 当前循环记录类型 直接读取对应列名就可以获取值
  3. 显式游标属性：
    1. %found 布尔型属性，当最近一次读记录时成功返回,则值为 true
    2. %notfound布尔型属性，与%found相反
    3. %isopen 布尔型属性，当游标已打开时返回 true
    4. %rowcount 数字型属性，返回已从游标中读取的记录数

2. 隐式游标：​

  1. 显式游标主要是用于对查询语句的处理，尤其是在查询结果为多条记录的情况下；
  2. 而对于非查询语句，如修改、删除操作，则由 ORACLE 系统自动地为这些操作设置游标并创建其工作区，这些由系统隐含创建的游标称为隐式游标，隐式游标的名字为 SQL
  3. 用户只能通过隐式游标的相关属性，来完成相应的操作
  4. 隐式游标属性：
    1. SQL%found 布尔型属性,当最近一次读记录时成功返回，则值为 true
    2. SQL%notfound 布尔型属性,与%found 相反
    3. SQL%rowcount 数字型属性, 返回已从游标中读取得记录数
    4. SQL%isopen 布尔型属性, 取值总是 false
  5. SQL 命令执行完毕立即关闭隐式游标

3. 动态游标：​

动态游标分为弱类型动态游标和强类型动态游标

--强类型动态游标类型
type 游标类型名 is ref cursor return emp%rowtype; -- 一般配合 记录类型

--弱类型游标类型
type 游标类型名 is ref cursor;

--类型动态游标系统自带了一个直接使用即可: sys_refcursor
refcur_emp sys_refcursor;
使用 open 打开动态游标并给其绑定结果集
open 动态游标名 for sql语句;

注意：

  1. 类型动态游标只能配合静态sql语句使用 无法和动态sql配合使用
  2. 动态游标无法使用for循环打开
