alter

1. alter基础使用：​

alter 语句 用于改动对象

1. 添加列
alter table 表名 add 列名 类型

2. 修改列
alter table 表名 modify 列名 类型

3. 删除列
alter table 表名 drop column 列名

4. 修改列名
alter table 表明 rename column 旧列名 to 新列名

5. 添加约束
alter table 表名 add constraint 约束名 约束关键字(列名)

6. 添加非空约束
alter table 表名 modify 列名 not null

7. 删除约束
alter table 表名 drop [novalidate] constraint 约束名

8. 禁用约束
alter table 表名 disable [validate] constraint 约束名

9. 启用约束
alter table 表名 enable

2. 注意：​

  1. 禁用和启动约束可选值：

    1. 禁用约束：

    disable (默认novalidate)：关闭约束,删除索引,可以对约束列的数据进行修改等操作 disable validate：关闭约束,删除索引,不能对表进行 插入/更新/删除等操作

    1. 启动约束：

    enable (默认validate)：启用约束,创建索引,对已有及新加入的数据执行约束 enable novalidate：启用约束,创建索引,仅对新加入的数据强制执行约束,而不管表中的现有数据

  2. 列的约束参考: 约束表
